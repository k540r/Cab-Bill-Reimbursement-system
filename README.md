# 🚖 Cab Bill Reimbursement System

A backend-driven web application designed to automate and manage employee cab expense claims with a structured approval workflow, role-based access control, and project management features.

---

## 📌 Overview

The system allows employees to submit cab reimbursement requests, which are reviewed and processed by admins and managers. It ensures transparency, secure access, and efficient tracking of expenses and project-related activities.

---

## 🏗️ Architecture

* Layered Architecture: Controller → Service → Repository
* RESTful API Design
* MVC Pattern Implementation
* Separation of concerns for scalability and maintainability

---

## 👥 User Roles

### 👤 Employee

* Submit cab bill reimbursement requests
* View request status and history
* Check assigned projects and managers
* View client visit discussions by team members

---

### 👨‍💼 Manager

* Manage multiple projects
* Add/remove employees from projects
* View project details and team activities

---

### 🛡️ Admin

* Review and process cab bill requests
* Approve / Reject / Delete requests
* Provide rejection reasons
* Monitor overall system activity

---

## ✨ Key Features

* Role-Based Access Control (Admin / Manager / Employee)
* Cab Bill Submission & Approval Workflow
* Expense Tracking & History Management
* Project & Team Management System
* Client Visit & Discussion Tracking
* Backend Validation & Exception Handling
* Secure and structured data flow

---

## 🔐 Security

* Role-based authorization implemented
* Secure API handling with controlled access
* Input validation to prevent invalid data

---

## 📊 Workflow

1. Employee submits cab bill request
2. Admin reviews the request
3. Request is approved / rejected with reason
4. Employee can track status and history
