The Cab Bill Reimbursement System is a backend application designed to automate and manage employee cab expense claims. Employees can submit reimbursement requests, while admins can review, approve, or reject them. The system ensures structured workflow, role-based access, and efficient expense tracking.
**Architecture**
Layered Architecture Controller → Service → Repository
RESTful API design
MVC pattern implementation**
**Tech Stack**
Backend: Java, Spring Boot, Spring MVC
ORM: Hibernate / JPA
Database: PostgreSQL
Build Tool: Maven
API Testing: Postman
**Key Features**
Role-Based Access Control (Admin / Employee)
Submit Cab Bill Requests
Approval/Rejection Workflow
Expense Tracking & History
Employee & Project Management
Backend Validation & Exception Handling
